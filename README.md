Investment API


## Investment API - веб-сервис для управления инвестиционными операциями и расчета стоимости торговых операций с ценными бумагами.

## 1. Решаемая задача: Сервис предоставляет инструменты для расчета стоимости инвестиционных операций, отслеживания истории покупок и установки автоматических триггеров для дополнительного выкупа акций при достижении целевых цен.




## 2. Основные функции сервиса

2.1. Управление ценными бумагами.

Просмотр доступных бумаг - получение списка акций с текущими ценами

2.2. Калькулятор инвестиций.

Расчет полной стоимости - автоматический учет количества акций, цены и комиссий

Планирование операций - предварительный расчет перед совершением сделки

2.3. Управление операциями.

История покупок - отслеживание всех совершенных операций

Детализация сделок - полная информация по каждой операции

2.4. Система триггеров.

Установка целевых цен - задание цен для автоматического выкупа

Отслеживание активных ордеров - мониторинг неисполненных триггеров

Уведомления о срабатывании - автоматическое определение достижения целевых цен



## 3. Архитектурные особенности

3.1. Backend технологический стек.

Платформа: ASP.NET Core 6.0

Язык программирования: C#

Архитектура: REST API

3.2. Frontend компоненты.

Интерфейс: HTML5 + CSS3 + JavaScript
